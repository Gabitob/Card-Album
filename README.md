## 🚀 Sobre o Projeto

O **Motta Card Album** é uma aplicação web completa de portfólio projetada para colecionadores de cartas de anime e jogos digitais. O sistema permite que o usuário gerencie sua própria coleção em uma interface sofisticada em modo escuro, oferecendo suporte a múltiplas páginas de exibição, upload direto de novas cartas e salvamento automático em nuvem por meio do ecossistema do Firebase.

## ✨ Funcionalidades Principais

* **Persistência com Firebase:** Integração direta com banco de dados NoSQL na nuvem para garantir que os cards salvos fiquem sincronizados e persistam mesmo após atualizar a página.
* **Sistema de Autenticação (Login/Sair):** Botão estruturado no cabeçalho superior para controle de sessão do usuário e proteção da coleção.
* **Navegação por Páginas Dinâmicas:** Paginação fluida e interativa (Página 1, 2, 3...) no topo da grade, permitindo expandir e organizar o álbum de forma limpa.
* **Upload por Drag-and-Drop:** Área dedicada para arrastar e soltar imagens locais ou clicar para selecionar arquivos no upload de novos cards.
* **Grade de Cards:** Layout em grid que exibe cartas estilizadas.
* 
## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto combina o poder do front-end nativo com serviços em nuvem (BaaS):

* **Firebase (Firestore / Auth / Storage):** Banco de dados em tempo real para armazenamento dos metadados das cartas, controle de sessão de usuários e hospedagem de imagens de upload.
* **JavaScript (ES6+):** Motor assíncrono do projeto, lidando com requisições à API do Firebase, lógica de troca de páginas e manipulação de eventos do DOM.
* **CSS3:** Identidade visual premium com tema escuro (Dark Mode), paleta neon reativa para indicar o nível de raridade dos cards e design responsivo baseado em CSS Grid.
* **HTML5:** Estrutura base da interface, incluindo formulários ocultos, contêineres de cards e botões funcionais de salvamento.
